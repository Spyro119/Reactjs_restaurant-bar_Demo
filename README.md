<!-- GETTING STARTED -->
## Getting started

### Prerequisites

- nodejs
- react
- typescript
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Spyro119/Reactjs_restaurant-bar_Demo/
   ```
2. Install dependencies
    ```sh
    npm i
    ```
3. Run the app 
    ```sh
    npm start
    ```
